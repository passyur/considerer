### pre-config

maybe [make a venv](https://docs.python.org/3/library/venv.html#module-venv)

```bash
pip install -r requirements.txt
```

```bash
cp .env.example .env
```
put your credentials in `.env` only

assuming internet enabled machine

python main.py
